Rock–Paper–Scissors Game Application

* This is a Python-based "Rock–Paper–Scissors" game application where the user plays against the computer.
* The game uses important programming concepts such as lists, dictionaries, functions, loops, exception handling, modules, and object-oriented programming.

Features:

* User vs Computer gameplay
* Random computer moves
* Score tracking
* Input validation
* Clean class-based code
* Modular structure (separate logic file)

Project Structure:

RockPaperScissors/
│
├── game_logic.py     # Contains class and game rules
├── main.py           # Runs the game
└── README.md         # Project documentation

How to Run:

1. Install Python 3
2. Download the project folder
3. Open terminal/cmd inside the folder
4. Run: python main.py

Requirements:

* Python 3.x
* No external libraries required


Game Rules:

* Rock beats Scissors
* Paper beats Rock
* Scissors beat Paper
